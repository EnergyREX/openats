# EnergyREX's OpenATS

## About
This project is an ATS system thought to be Open Source. This version is thought only for applicants. Refer to recruiter branch to the recruiter version.

## Stack
Main stack is splitted by two sides, backend and frontend.

**Backend**: Fastify, DrizzleORM, BullMQ... etc. Arch DDD + CQRS.
**Frontend**: React.js + MaterialUI.
**Infrastructure & Deploy**: Docker (nginx, node.js), Redis, PostgreSQL

## Features

[ ] - Reading a CV, PDF, DOCX formats only.
[ ] - Tracking your own applications and if you may match the offer. (**Manual**)
[ ] - AI Evaluation
[ ] - Dashboard... etc.
